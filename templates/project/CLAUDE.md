# Claude Code 프로젝트 지침

이 프로젝트의 구조·명령어·규칙은 **AGENTS.md 를 따른다.** 여기에는 Claude 에만 해당하는 것만 적는다.
공통 내용을 여기에 중복해서 적지 말 것 — 두 곳이 어긋나면 어느 쪽이 맞는지 알 수 없게 된다.

## 이 프로젝트에서

- <예: E2E 는 Playwright 로 작성한다. Cypress 코드를 새로 만들지 않는다.>
- <예: Supabase 마이그레이션은 확인 없이 수정하지 않는다.>
